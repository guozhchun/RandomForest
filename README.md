# RandomForest

This is a random forest algorithm written by C++. It is used for the non-linear classification.

## Files

`RandomForest.cpp` ：the source code

`train.csv`：the training data file

`test.csv`：the predict data file

`output.csv`：the predict result. It may be different when running the program again. It will be created if the file is not exist and run the program in first time.

# How to run

use the `Visual Studio` to open the project by double click `RandomForest.sln` file. Or make a new project and the import the `RandomForest.cpp`、`train.csv`、`test.csv` and then run the project to get a result.
